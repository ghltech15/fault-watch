# Plan 01-02: Layout Restructure & Visual Hierarchy

## Objective

Restructure the dashboard layout for better visual hierarchy and reduced clutter. Make key information instantly scannable.

**Purpose:** Transform cluttered data display into clear, prioritized information hierarchy.
**Output:** Restructured tabs with news-style layout and clear hierarchy.

**Current State:**
- 7 tabs all with similar density
- Dashboard tab has too much information
- No clear visual hierarchy (everything looks equally important)
- Metrics scattered across columns without grouping

**Target State:**
- Clear primary/secondary/tertiary information levels
- Dashboard shows only critical metrics prominently
- Details available on demand (expanders, secondary tabs)
- News-style "above the fold" priority content
